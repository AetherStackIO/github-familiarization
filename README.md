## GitHub Familiarization Project

# GitHub Familiarization Project: Learning Journal

## Objective

Learn the complete Git and GitHub workflow by maintaining a simple learning journal.

No programming required.

---

## Project Structure

learning-journal/

├── README.md

├── notes/

│   ├── day1.md

│   ├── day2.md

│   └── day3.md

└── goals.md

---

## Step 1: Create Repository

Repository Name:

learning-journal

Description:

My Git and GitHub learning journey.

---

## Step 2: Create README.md

```md
# Learning Journal

This repository contains my notes while learning Git and GitHub.

## Goals

- Learn Git basics
- Practice commits
- Practice pull requests
- Learn branching
```

---

## Step 3: Create goals.md

```md
# Goals

- Learn git init
- Learn git clone
- Learn git add
- Learn git commit
- Learn git push
- Learn pull requests
```

## Step 5: Commit Changes

